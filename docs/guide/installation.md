# Installation

## From the Unity Asset Store

1. Purchase or download FludeX from the Unity Asset Store
2. Open **Package Manager → My Assets**
3. Find **FludeX** → click **Download**, then **Import**
4. App UI installs automatically as a dependency — no additional setup needed

## Dependency

FludeX depends on `com.unity.dt.app-ui` version **2.1.6**. This package is declared as a UPM dependency and installs automatically when you import FludeX — no manual step is required.
